A Django template project for beginners

To create a new page

  • Set a new path on /config/urls.py
  • Create a view on /project/views.py
  • Create a template on /project/templates

To create a new model

  • Create a new class on /project/models.py
  • Run on console 
    python manage.py makemigrations
  • Run on console 
    python manage.py migrate

To run the webserver

  • Run on console 
    python manage.py runserver

To install requirements

  • All project packages: 
    pip install -r requirements
  • A single package: 
    pip install package

To create a superuser

  • Run on console 
    python manage.py createsuperuser
  • Follow the instructions after the command
